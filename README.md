# Alberto Álvarez-Álvarez academic website

Static academic website starter for GitHub Pages.

## How to use
1. Create a GitHub repository named `yourusername.github.io`.
2. Upload all these files.
3. In GitHub, go to Settings > Pages and publish from the main branch.
4. Replace the photo placeholder with `assets/img/profile.jpg`.
5. Add your full CV PDF and link it from `cv.html`.
